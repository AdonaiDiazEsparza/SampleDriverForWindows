# ATRAPA EL EVENTO
El [código](Source.cpp) creado para este ejemplo, recibe por medio de la función ```NotifyForAProcessCreation``` la creación de algún proceso en Windows enseñando que el ```id``` de el proceso y su nombre.

Por el momento se genera este ejmplo para ver en debug o con el programa [DebugView](https://learn.microsoft.com/en-us/sysinternals/downloads/debugview) de windows que nos ayudara a debbuguear cada uno de los logs de las pruebas
